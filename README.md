# Student Management System

A Java console-based application developed to practice Object-Oriented Programming (OOP) concepts.

## Features

- Add Student
- Update Student
- Delete Student
- Search Student
- View All Students

## Project Structure

- Model
- DAO
- Service
- Utility
- Main

## Technologies

- Java
- Eclipse
- Git
- GitHub

## Concepts Used

- Classes and Objects
- Encapsulation
- Collections
- Exception Handling
